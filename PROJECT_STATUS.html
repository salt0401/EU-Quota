MEPS · EU/UK Steel Quota Tracker

The quota tracker now runs itself.

Live — updates daily at 05:30 UTC Status as of 6 July 2026 Ready ahead of the 17 July deadline

Since 1 July the EU and UK have run brand-new steel import quota systems, and the tracker has been fully rebuilt around them. More importantly for you: you no longer run the scraper. A GitHub server collects every quota every morning, keeps a day-by-day history, and all you do is double-click one small program to fetch the latest files.

What you do now

Double-click MEPS_Quota_Downloader.exe — that's it.

It's a single small file (no installation, no login). In a few seconds it fetches the latest data into data\output\<today's date>\ next to the program, and tells you when the data was generated. Run it whenever you want fresh numbers — daily, weekly, or right before a meeting.

You receiveWhat it is
MEPS_Quota_Update_latest.xlsx The familiar customer report (EU + UK tabs, slicers), for the latest day.
Quota_History.xlsx One row per quota per day, EU and UK sheets — this grows every day, so you can filter any quota and watch it drain over time. Built for your analysis.
quota_history.csv The same history as plain data, ready for pivot tables, Power Query, or Python.

How it works

The scraping moved from your desktop to GitHub's servers (free for public projects).

1

GitHub scrapes, every morning

At 05:30 UTC a GitHub Actions job queries all 283 EU quotas on TARIC and all 75 UK quotas on the UK tariff API. A full run takes about 90 seconds.

2

Results are published

The report and the growing history are published to the public repository — each day adds 358 rows to the history, one per quota.

3

You download

The downloader fetches the published files over plain public links. No account, no token, no VPN — if you have internet, it works.

How we got here

  • 1 July
    The old steel safeguards expired. New regimes began: EU Regulation 2026/1384 (new order numbers, MFN + FTA quota parts) and the UK steel trade measure (volumes cut 51%, Ukraine exempt).
  • 2 July
    Your brief arrived with the EU regulation attached. The UK announcement (Tables 3 & 4) was tracked down independently and cross-checked against the live HMRC API.
  • 6 July
    Everything shipped. All 358 new order numbers wired in and verified against the regulations to the tonne; the report template rewritten for the new rules (50% duty, new quota types); several long-standing data bugs fixed — including one that hid "awaiting allocation" volumes, which today decide whether a quota is really open.
  • 6 July
    Daily automation switched on and stress-tested: 188 automated tests, an adversarial review that fixed 17 further weaknesses, and three full live runs with zero failed quotas.

Good to know

  • Freshness The downloader shows the data's date and warns if the daily update has stopped (a weekend gap is normal for the UK; TARIC moves daily).
  • Failures If a morning run fails, it automatically opens a GitHub issue and your downloads simply keep serving the last good day — nothing breaks quietly.
  • 1 October Quarter turn — a small input-file update on the maintainer's side. The pipeline self-checks against stale-quarter data in the meantime.
  • January 2027 The EU implementing regulation runs to 31 December 2026; its renewal may bring new order numbers. The system is designed to fail loudly (not publish nonsense) when that lands.
